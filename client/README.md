# SuperMall - Rural E-commerce Platform

SuperMall is a comprehensive e-commerce platform designed to connect rural merchants with digital markets, enabling them to showcase and sell their products online.

## Features

### User Features
- Browse products by category
- View detailed product information
- Filter and search products
- Compare multiple products
- View shop information and locations
- Browse active offers and deals

### Admin Features
- Manage shops and merchants
- Add and update products
- Create and manage offers
- Organize categories
- Manage floor layouts

## Tech Stack

**Frontend:**
- React 18
- React Router v6
- Axios
- React Icons
- React Toastify

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Express Validator
